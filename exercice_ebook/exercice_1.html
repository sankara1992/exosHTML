Question 1
La déclaration DOCTYPE est-elle obligatoire ? Où doit-elle être placée ? Quelles informations contient-elle ?
Réponse 1 : Elle est plus recommandé, donc obligatoire et elle se place en tête du code devant la balise <html>.Elle contient le nom de l'élément racine
Question 2
Que faut-il faire opur que le code HTML soit valide ?
Réponse 2 : Il faut respecter l'inclusion ou l'imbrication des éléments et les attributs à chacun d'entre eux et ensuite verifier avec le validateur du W3C
Question 3
À quel emplacement est defini le jeu de caractères utilisé dans le document ?
Réponse 3 : le jeu de caractère est précisé dans un élément <meta> meta charset="UTF-8">
Question 4
Où est déclaré le nom de l'élément racine du document ? Quel est son rôle et son nom ?
Réponse 4 : le nom de lélément racine est donné dans DOCTYPE il s'agit de html ; c'est l'élément qui inclus tous les autres donc il est au sommet de la hiérarchie
Question 5
Quels sont les éléments enfants de l'élément HTML ?
Réponse 5 :les enfants (de premier de niveau) sont <Head>Body>
Question 6
À quoi sert l'élement <head> ? À quoi sert-il ?
Réponse 6 :L'élément head est l'én-tête du document, il n'est pas visible dans le navigateur (sauf le title) : il contient BASE, LINK, META, SCRIPT, STYLE, TITLE. seuls les éléments BASE et TITLE ne doivent être utilisés qu'une seule fois
Question 7
Quel élement est obligatoire dans l'élément <head> ? À quoi sert-il ?
Réponse 7 : seul l'élément TITLE est obligatoire. C'est le titre dans l'onglet
Question 8
Écrire le code nécessaire à la liaison d'une feuille de style avec un document
Réponse 8 : <link rel="stylesheet" href="css/style.css"
Question 9
Écrire le code nécessaire à la liaison d'un document externe contenant des scripts JavaScript avec un document
Réponse 9 : <script src="js/mes_scripts.js"></script>

Réponse 10 : <meta name="keywords" content="Exercices eBook, HTML, patrick isola">
