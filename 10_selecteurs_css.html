Aide mémoire sélecteurs css 3

Les sélecteurs css

  • *: sélectionne Tous les éléments de la page HTML
  • .ma-jolie-class : sélectionne tous les éléments ayant le nom de class "ma-jolie-class"
  • #mon-unique-id : sélectionne tous les éléments ayant le nom d'ID "mon-unique-id"
  • div, aside, article : sélectionne toutes les balises <div>, <aside>, et <article>
  • nav a : sélectionne les liens <a> imbriqués dnas la balises
  • nav a:hover : sélectionne tous les liens de la nav et active une animation au survol de la souris

propiétés xcss utilisées dans cette page

Décrivez leur usage valeurs possibles

  1. propriétés pour le texte
    • font-family: famille de typo dans un ordre de propiété
    • font-size : taille de la typo, des caractères
    • text-align:alignement du texte, à gauche à droite ou au centre
    • line-height: interlignage, espace entre les lignes
    • text-decoration : pour "decorer" le texte en ajoutant une ligne en dessous
    • color : couleur de la typo, de la font ou du texte
  2. propriétés des éléments
    • background: signifie arrière-plan d'un élément html
    • height: hauteur de l'élément
    • width: largeur de l'élément : tous les deux par rapport à la taille de la fenêtre si on indique une valeur proportionnelle en % ou en em
    • left: position à partir de la gauche de la fenêtre du navigateur
    • margin: marge extérieure d'un élément
    • padding:marge intérieur d'un élément
    • float :permet de sortir un élément du flux / effet flottant
    • position: position de l'élément dans la page
    • top: position par rapport au haut de la fenêtre
    • transition : temps de transition lors d'une animation

propriétés Css a rechercher

Rechercheez les propriétés ci-dessous, faites des tests avec ces propriétés sur cette page et depuis la console de débuggage testez les valeurs qu'elles peuvent prendre

  • font-weight : permet de définir la graisse pour le texte
  • letter-spacing : définit l'interlettre utilisée pour les caractères qui composent le texte
  • word-spacing : définit la règle d'espacement utilisée entre les balises et entre les mots
  • text-shadow : ajoute des ombres au texte, elle accepte une liste à appliquer au texte et aux décorations de l'élément
  • background-image : permet de définir une ou plusieurs images comme arrière-plan pour un élément
  • background-size : définit la taille des images d'arrière-plan popur l'élément
  • opacity : définit la transparence d'un élément, autrment dit elle définit le dégré de visibilité de l'arrière-plan sur lequel est placé l'élément
  • border-style : une est propriété raccourcie qui permet de définir le style des lignes utilisées pour les bordures des quatres cotés de la boite d'un élément
  • box-shadow : ajoute des ombres à la boite d'un élément via une liste d'ombres separées par des virgules
  • list-style-image : définit l'image utilisée comme puce devant les éléments de listes
  • border : est une propriété raccourcie qui permet de définir les propriétés liées à la bordure, elle est peut-etre utilisée pour définir les valeurs (border-width, border-style, border-color)
  • border-radius : permet de définir des coins arrondis pour la bordure d'un élément
  • border-width : est une propriété raccourcie qui définit la largeur de la bordure d'un élément
  • font-variant :est une propriété raccourcie représentant les propriétés : font-variant-caps, font-variant-numéric...
  • background-color : permet de définir la couleur utilisée oour l'arrière-plan d'un élément (celle-ci peut-etre une couleur transparente
  • !important :permet de spécifier que la propriété est la plus importante dans votre feuille Css en lui donnant un poids plus important
