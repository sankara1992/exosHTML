Les sélecteurs css

  • *: sélectionne tous les éléments de la page HTML
  • .ma-jolie-class :sélectionne tous les éléments ayent le nom de class "ma-jolie-class
  • #mon-unique-id :sélectionne tous les éléments ayant le nom d'ID "mon-unique-id
  • div, aside, article :sélectionne toutes les balises <div>, <aside>, et <article
  • nav a
  • nav a:hover
